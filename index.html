Step-by-Step Guide to Create a Vite Project

Open Your Terminal/Command Prompt:

step1:-

npm create vite@latest my-vite-app

step2:-

Replace my-vite-app with your desired project

Choose a Framework:

After running the command, you will be prompted to select a framework. You can choose from options like:

React

Vue

Svelte

Vanilla

step3:-

cd my-vite-app

step4:-

npm install

step5:-

npm run dev

This will start the development server, and you can access your app at http://localhost:5173 (or another port if specified).
